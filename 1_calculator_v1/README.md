# Calculator v1

A simple command-line calculator written in Python that allows users to perform basic arithmetic operations.

---

## Goal

Learn the fundamentals of programming logic through a basic calculator:

- User input handling
- Conditions
- Loops
- Functions
- Error management

---

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation
- Division by zero protection
- Continuous calculations (loop system)

---

## Requirements

- Python 3.x (tested on Python 3.7+)

---

## Installation

No external dependencies required. Simply clone and run!

```bash
git clone <your-repo-url>
cd 1_calculator_v1
```

---

## How to Run

```bash
python main.py
```
---

## Example Usage

```
$ python main.py

=== Welcome to the mini-calculator ===

Enter a value for a:
>> 10

Choose an operation (+, -, *, /):
>> /

Enter a value for b:
>> 2

Result: 10.0 / 2.0 = 5.0

Do you want to perform another calculation? (y/n): n

Thanks for using the calculator. Goodbye!
```
---

## Project Level

Beginner

---

## Limitations & Known Issues

- Only supports basic arithmetic (+, -, *, /)
- No support for parentheses, exponents, or complex expressions
- Input is limited to single operations
- No history of calculations
- No GUI
