# python-path-beginner

A structured Python practice repository focused on building and
demonstrating core programming fundamentals through progressively
challenging projects.

This repository is not designed as a teaching course, but as a
**personal training path** to strengthen Python skills, improve
programming logic, and demonstrate practical mastery of core concepts
through real implementations.

------------------------------------------------------------------------

## Purpose

The goal of this repository is to:

-   Practice Python fundamentals in real projects\
-   Build strong programming logic\
-   Develop clean coding habits\
-   Improve problem-solving skills\
-   Create a structured learning progression\
-   Demonstrate technical mastery, not just theory

This is a **practice-driven repository**, focused on doing, building,
and understanding --- not tutorials.

------------------------------------------------------------------------

## Project Structure

Projects are organized by increasing difficulty:

    Beginner/
    ├── 1_calculator_v1
    ├── 2_text_analyzer_v1
    └── 3_phonebook_v1

Each project is self-contained and has its own documentation.

------------------------------------------------------------------------

##  Overview

### 1. Calculator v1

Concepts: - User input - Conditions - Loops - Functions - Error

------------------------------------------------------------------------

### 2. Text Analyzer v1

Frequency analysis - Basic statistics - Structured output

------------------------------------------------------------------------

### 3. Phonebook v1

**Focus:** Structured programming & data modeling\
Concepts: - Classes & objects - Data structures - File handling - CRUD

------------------------------------------------------------------------

## Progression Logic

Each project increases in complexity:

1.  **Calculator** → basic logic and control flow\
2.  **Text Analyzer** → structured data processing\
3.  **Phonebook** → system thinking, persistence, and architecture

The goal is to move from simple logic → structured logic → system logic.
